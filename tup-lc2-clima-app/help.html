Ayuda Clima App v.1.0

Esta App te permite consultar el clima de una ciudad en particular utilizando la API de OpenWeatherMap.

Para poder consultar el clima, primero debera Agregar una ciudad y luego seleccioanra la ciudad en cuestion desde la pagina Clima.
